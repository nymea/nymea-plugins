# Home Connect

Connects your Home Connect home appliances to nymea.

## Supported Things

Home connected home appliances, like:
* Oven
* Dishwasher
* Coffee maker
* Dryer
* Fridge
* Washer
* Cook Top
* Hood

## Requirements

* The package “nymea-plugin-homeconnect” must be installed
* Internet connection
* Home connect account
